To Run the Application:-
    * Go to the root folder
    * Type this command in the terminal `python main.py`
    * Three challenges are available to run
        1) Predict outcome
        2) Predict outcome with comment
        3) Super Over
    * We can select the challenge by corresponding number and proceed
    * All the settings of the match can be found in the `match_settings.py` file
    * We can edit the match settings and run the application
    * In `outcome_chart.json` all probabilities are mapped
    
To run all the tests:-
    From the root folder type this command `python -m unittest discover tests`
